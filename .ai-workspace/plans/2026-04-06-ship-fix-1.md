# Fix: buildStageList stage name mismatch

## Problem
`buildStageList` uses `tierLabel ? "master " : ""` for critic/corrector stage names, but only the master tier uses `runMasterCritic`/`runMasterCorrector`. Phase and update tiers use standard `runCritic`/`runCorrector` which report stages without "master" prefix.

## Fix
Change condition to only use "master " when `documentTier === "master"`.
